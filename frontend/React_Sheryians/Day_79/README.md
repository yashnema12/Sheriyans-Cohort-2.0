React-router-DOM

- install it
- import {BroswerRouter} in main.jsx
- wrap <App/> inside BrowserRouter 
- make routes (collection of all routes) inside <Routes> <Routes/> make Route
- mention path in <Route/> and elements
- use <Link> tag to redirect any specific route
- in navbar use <Link> by react-router-dom to redirect any specific route
- nested routes can be write as - path="/product/men"
- Dynamic routes can be write as - path="/product/:id"

- you can access id using useParams() hook inside that element you rendered in your dynamic route
